最近安装面板后端的总结

2020-08-05 2 min read

###主要碰到的问题：

  • pip版本过新导致了部分安装内容版本太高 与一键脚本不兼容
  • 多数作者删库跑路
  • 安全组限制

###记录一下最近常用的指令和脚本：
使用旧版本pip

wget https://pypi.python.org/packages/source/p/pip/pip-9.0.1.tar.gz --no-check-certificate
tar -xzvf pip-9.0.1.tar.gz
cd pip-9.0.1
python setup.py install
pip install cymysql==0.9.4

安装所有依赖

pip install -r requirements.txt

一键安装BBR Plus
目前实测下来bbrplus优于锐速

wget --no-check-certificate -O tcp.sh https://github.com/cx9208/Linux-NetSpeed/raw/master/tcp.sh && chmod +x tcp.sh && ./tcp.sh

来自秋水大佬的一键跑分脚本

wget -qO- bench.sh | bash

一键后端脚本

yum install wget -y && wget https://raw.githubusercontent.com/Toxic-Cat/Airport-toolkit/master/ssr_node_c7.sh && chmod +x ssr_node_c7.sh && ./ssr_node_c7.sh

安装目录

/soft/shadowsocks

查看log指令

./tail.sh

监控脚本
登录地址

wget --no-check-certificate https://auto.feipig.cc/install_AutoClient.sh -O install_AutoClient.sh && bash install_AutoClient.sh A.B.C.D

ABCD需替换成对应ip

逗比backup无脑一键ssr脚本

wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh
